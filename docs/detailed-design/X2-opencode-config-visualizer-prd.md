# [R12] OpenCode 配置可视化 PRD（当前实现对齐版）

## 背景
项目已经具备 OpenCode 相关配置读写能力，需要通过可视化方式降低使用门槛并减少配置错误。

## 目标
- 通过可视化预览降低配置理解成本
- 保持与项目级配置文件结构一致
- 与蜂群配置应用流程兼容

## 功能需求
- 预览能力：在蜂群预览中展示 OpenCode 配置结构（只读）
- 编辑能力：在项目创建与项目设置中编辑配置文本
- 保存能力：应用时执行基础校验，失败时返回可读错误

## 数据契约
- 输入输出与 `oh-my-opencode.jsonc`、`opencode.json` 结构保持一致
- 保存后写入项目目录供执行模块消费

## 验收要求
- 可完成读取、预览、文本编辑、保存全链路
- 保存后重新读取结果一致
- 不要求独立可视化编辑器页面

## 非目标
- 不包含 UI 图示、组件布局说明、视觉还原要求
