# [R3] 任务派发与 Agent 执行需求（当前实现对齐版）

## 范围
- 从任务触发执行会话
- 执行日志流与工具流展示
- Follow-up 与 Git 操作协同

## 功能需求
- 支持创建执行会话并绑定任务上下文
- 支持执行过程实时状态同步
- 支持标准化日志流解析与展示
- 支持 Follow-up 消息继续执行
- 支持执行结束后的 Git 操作（分支、提交、推送、合并相关流程）

## 执行器要求
- 支持多执行器接入与可用性发现
- 统一执行器输入输出结构，便于前端渲染
- 支持命令/工具调用结果可追踪

## 集成要求
- 前端模块：`features/agent-execution`
- 后端模块：`infra/tauri` 执行与会话能力

## 非目标
- 不保留 UI 线框图、页面布局说明、回填任务清单
