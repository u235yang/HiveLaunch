# [R9] 远程访问 PRD（当前实现对齐版）

## 产品目标
- 在不改变本地优先架构的前提下，提供移动端远程访问能力
- 支持不同网络环境下的稳定连接

## 功能范围

### 1. 配对与设备管理
- 支持桌面端生成配对信息
- 支持移动端通过扫码或手动输入完成配对
- 支持已配对设备管理

### 2. 连接模式
- 支持局域网直连模式
- 支持 Relay 中转模式
- 支持模式切换与连接状态感知

### 3. 远程访问设置
- 支持在设置模块启停远程访问
- 支持查看当前连接状态与关键参数
- 支持失败原因反馈与重试

## 技术与安全约束
- 核心业务 API 保持统一 HTTP 通道
- 本地服务仍为业务处理中心
- 中转链路仅负责连接转发与会话维持
- 连接过程必须具备基础认证与加密能力

## 验收标准
- 能完成配对、连接、断开、重连完整流程
- 在直连与 Relay 模式下均可访问任务执行状态
- 连接异常可被识别并提示

## 非目标
- 不保留 UI 资源路径、页面草图与视觉描述
- 不包含未落地的扩展运营功能
